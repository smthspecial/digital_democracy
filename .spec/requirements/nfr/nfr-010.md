## Description

The system shall remain understandable to an average citizen without specialized education, so participation does not depend on intermediaries.

## Metric

- Target: Average citizen can understand eligibility, proposals, budgets, and how to vote unaided.
- Measurement: Usability testing with representative citizens; comprehension and task-completion rates.
