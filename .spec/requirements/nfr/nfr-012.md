## Description

The system's governance software and technical specifications shall be open source and openly published, so the platform itself is publicly auditable and not captured by its operators.

## Metric

- Target: All governance-critical code and specifications publicly available and reproducible.
- Measurement: Public repository availability; reproducible builds; specification completeness review.
