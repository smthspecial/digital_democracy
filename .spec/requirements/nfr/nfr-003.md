## Description

The system shall provide cryptographic verifiability so that anyone can confirm votes were recorded and tallied correctly, without trusting operators.

## Metric

- Target: 100% of tallies independently reproducible from public verification data.
- Measurement: Public verifier tooling reproduces tallies; universal/individual verifiability tests pass.

## Traceability
