## Description

The system shall make all identity creation events auditable by independent parties.

## Acceptance Criteria

- [ ] Every identity creation event is recorded in an append-only log
- [ ] Independent auditors can review creation events without accessing private data
- [ ] Anomalous creation patterns are detectable
