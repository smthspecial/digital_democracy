## Description

The system shall distribute governance work by assigning each citizen a small number of civic duties (reviews, audits) prioritized by sphere of impact, competency domains, workload balancing, and random selection.

## Acceptance Criteria

- [ ] Each citizen receives a limited number of assignments
- [ ] Assignments are weighted by impact, competency, workload, and randomness
- [ ] Assignment frequency adjusts automatically to balance load
