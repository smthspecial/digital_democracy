## Description

The system shall allow impact-scope assignments to be challenged; citizens may request review.

## Acceptance Criteria

- [ ] Any eligible citizen can file a scope challenge
- [ ] Challenges are tracked with status and outcome
- [ ] A pending challenge can pause the affected voting phase where required
