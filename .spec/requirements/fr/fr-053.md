## Description

The system shall support valid exemptions (illness, disability, military service, caregiving, other verified circumstances) that pause civic obligations without penalty.

## Acceptance Criteria

- [ ] Citizens can claim verified exemptions
- [ ] Exemptions pause obligations without penalty
- [ ] Exemption records are auditable
