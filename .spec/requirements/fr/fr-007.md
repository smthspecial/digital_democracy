## Description

The system shall ensure no administrator can unilaterally disable citizen participation; multiple independent approvals shall be required.

## Acceptance Criteria

- [ ] Disabling an identity requires multiple independent approvals
- [ ] No single administrator role can complete the action alone
- [ ] Every disabling action is logged with all approvers
