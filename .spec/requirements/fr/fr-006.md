## Description

The system shall provide identity revocation procedures for death, loss of citizenship, and proven fraud.

## Acceptance Criteria

- [ ] Defined revocation workflows exist for each qualifying condition
- [ ] Revocation requires documented justification and is logged
- [ ] A revoked identity can no longer participate
