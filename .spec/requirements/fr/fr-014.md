## Description

The system shall classify proposals into six impact-scope levels (property, street/neighborhood, municipality, regional, national, constitutional) and assign authority to the smallest competent jurisdiction, escalating only when a problem exceeds local scope.

## Acceptance Criteria

- [ ] Each proposal is assigned one of the six defined scope levels
- [ ] Authority defaults to the smallest competent jurisdiction
- [ ] Escalation to a higher level requires a documented reason
- [ ] Impact is evaluated across geographic, economic, environmental, infrastructure, and legal dimensions
