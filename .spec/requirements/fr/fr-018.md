## Description

The system shall allow multiple competing proposals to address the same problem so citizens evaluate alternatives rather than a single government proposal.

## Acceptance Criteria

- [ ] Multiple proposals can be linked to one problem
- [ ] Competing proposals are presented side by side
- [ ] Citizens compare alternatives during deliberation and voting
