## Description

The system shall organize public budgets into categories so citizens allocate priorities within categories rather than competing for a single undifferentiated pool.

## Acceptance Criteria

- [ ] Budgets are divided into defined categories
- [ ] Allocation occurs within categories
- [ ] Partitioning reduces zero-sum framing and improves transparency
