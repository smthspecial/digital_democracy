## Description

The system shall make competency status non-permanent, requiring periodic review (e.g. every 2-5 years) and revoking active status from inactive contributors.

## Acceptance Criteria

- [ ] Competency status carries an expiry and review schedule
- [ ] Inactive contributors lose active status
- [ ] Re-validation is required to retain status
