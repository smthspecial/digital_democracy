## Description

The system shall give each submitted issue public visibility and allow citizens to support, comment, and provide additional evidence.

## Acceptance Criteria

- [ ] Submitted issues are publicly listed
- [ ] Citizens can support, comment on, and add evidence to an issue
- [ ] Support counts are visible and auditable
