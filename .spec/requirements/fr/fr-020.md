## Description

The system shall guarantee that no citizen receives additional voting rights based on education, wealth, profession, status, or credentials; all citizens remain politically equal.

## Acceptance Criteria

- [ ] Competency status never changes a citizen's voting weight
- [ ] Voting weight is identical for all eligible citizens
