## Description

The system shall verify residency eligibility for participation, supporting configurable minimum residency periods (e.g. 6-12 months).

## Acceptance Criteria

- [ ] Residency is verified against the proposal's jurisdiction
- [ ] Configurable minimum residency periods can be enforced per jurisdiction
- [ ] Residency status is re-checked at voting time
