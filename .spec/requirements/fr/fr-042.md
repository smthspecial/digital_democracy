## Description

The system shall apply thresholds matched to decision type: local simple majority; national infrastructure majority plus minimum participation; constitutional supermajority.

## Acceptance Criteria

- [ ] Threshold rules are defined per decision type
- [ ] National-scale decisions require a minimum participation quorum
- [ ] Constitutional decisions require a supermajority
