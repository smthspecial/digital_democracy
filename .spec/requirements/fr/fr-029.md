## Description

The system shall require proposals to begin with constraint identification, defining what any acceptable solution must satisfy before solutions are debated.

## Acceptance Criteria

- [ ] Each proposal records agreed constraints before solution design
- [ ] Solutions are evaluated against the agreed constraints
- [ ] Constraint sets are public
