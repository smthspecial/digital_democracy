## Description

The system shall provide a transparent expertise acquisition process: application, automated verification, public review, domain review, and recorded approval.

## Acceptance Criteria

- [ ] A citizen can apply for competency in a domain with evidence
- [ ] Automated verification of credentials runs where possible
- [ ] Applications are publicly reviewable before approval
- [ ] Approval records all evidence and reasoning
