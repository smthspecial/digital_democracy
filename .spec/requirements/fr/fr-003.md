## Description

The system shall cryptographically separate voting records from personal identity, verifying eligibility without exposing vote choices.

## Acceptance Criteria

- [ ] Eligibility can be confirmed without linking a voter to a ballot
- [ ] No stored record allows reconstruction of how an identified citizen voted
- [ ] Ballot secrecy is preserved end-to-end
