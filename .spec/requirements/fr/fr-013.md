## Description

The system shall allow citizens to participate in multiple jurisdictions simultaneously (neighborhood, city, region, nation).

## Acceptance Criteria

- [ ] A citizen can hold simultaneous memberships across nested jurisdictions
- [ ] Eligibility is evaluated per jurisdiction for each proposal
