## Description

The system shall allow any citizen to submit problems, opportunities, and improvement suggestions, including description, affected area, and supporting evidence.

## Acceptance Criteria

- [ ] Any citizen can create a problem entry
- [ ] Submission captures description, affected area, and evidence
- [ ] Submitted issues are assigned to a candidate impact scope
