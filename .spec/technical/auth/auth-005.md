## Description

A citizen assigned to monitor active project implementation. Oversight role holders track milestone completion, budget consumption, and contractor activity, and surface deviations to the audit pool and public. They observe and report; they do not control implementation method or logistics (FR-045).

## Acquisition

Generated by civic-duty-service (SRV-009) via DP-040 when a project enters `in_progress` status. Assignment scoped to a specific `project_id` via `civic_assignment.target_ref`.

## Capabilities (additive over AUTH-001)

| Capability | Condition | FR |
|-----------|-----------|-----|
| Report project milestone updates | Active governance_role (type=oversight), within assigned project | FR-046 |
| Submit outcome evaluation | Active term, assigned to the project's evaluation task | FR-048 |
| Flag budget deviation for audit review | Active term, assigned project | FR-047 |
| View non-public project implementation details | Active term, assigned project | FR-047 |

## Restrictions

- Cannot modify the implementation approach, select contractors, or override logistical decisions (FR-045).
- Observations are public reports; they cannot directly pause or cancel a project without routing through the audit and approval process.
- COI exclusion applies: excluded from oversight of projects where a personal financial or organizational conflict exists (FR-063).

## Term and rotation

- Role is project-scoped. Expires when the associated project reaches `status=completed` or `cancelled`. DP-050 enforces expiry.
