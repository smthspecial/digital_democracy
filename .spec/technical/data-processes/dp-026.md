## Description

Decrypts all `ballot` rows (threshold cryptography across distributed key holders), applies the session's `method` and `threshold_rule`, and writes the tally result to `vote_session`. Worker holds a distributed lock; exactly-once semantics enforced via session-level idempotency key. Enqueues DP-027.
