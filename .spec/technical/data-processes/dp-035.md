## Description

Monitors the set of `approval` rows for a given action. Checks that all three required types are satisfied (`citizen_supermajority`, `audit_confirmation`, `body_endorsement`) by independent role holders. On completion, either releases the action or enqueues DP-043 for protocol changes requiring delay.
