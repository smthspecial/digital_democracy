## Description

Atomically: (1) validates the blinded token is unused, (2) writes a `ballot` row with no `citizen_id`, (3) marks `eligibility_token.used=true`. The three steps are a single database transaction. Enqueues DP-041 if the citizen has an active delegation for the proposal's domain.
