## Description

Inserts a signed `reputation_record` delta row. Positive factors: accurate prediction, constructive contribution, disclosure, successful proposal. Negative factors: misinformation, undisclosed conflict, manipulation, fraud. Negative deltas require an upstream authoritative decision. Reputation never alters voting weight.
