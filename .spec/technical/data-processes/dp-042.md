## Description

Cascades a completed identity revocation: sets `delegation.revoked_at`, cancels open `civic_assignment` records, invalidates unused `eligibility_token` rows, suspends active `governance_role` records. All changes are emitted to DP-036. Requires prior multi-approval via DP-035 (FR-007).
