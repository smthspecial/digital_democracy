## Columns

| Column | Type | Description |
|--------|------|-------------|
| id | uuid | Primary key |
| vote_session_id | uuid | Session the citizen is eligible for |
| citizen_id | uuid | Eligible citizen |
| blinded_token_hash | text | Hash of a blind-signed token issued to the citizen |
| issued_at | timestamptz | Issuance time |
| used | boolean | Whether a ballot was cast with this token |

## Notes

Implements FR-003, FR-041. Proves a citizen MAY vote and voted, WITHOUT linking to ballot content. The blinded token is the only bridge to TBL-022 and cannot be joined back to the citizen (NFR-001).
