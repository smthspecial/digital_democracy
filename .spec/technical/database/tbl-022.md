## Columns

| Column | Type | Description |
|--------|------|-------------|
| id | uuid | Primary key |
| vote_session_id | uuid | Session this ballot belongs to |
| token_blind | text | Unblinded one-time token (NOT linkable to citizen) |
| encrypted_choice | text | Encrypted ranked/approval selection |
| verification_code | text | Code the voter uses to confirm inclusion |
| cast_at | timestamptz | Cast time |

## Notes

Implements FR-003, FR-004. DELIBERATELY has no citizen_id. Ballot secrecy: cannot be joined to eligibility_token's citizen (NFR-001, ADR-002).
