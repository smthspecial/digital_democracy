## Columns

| Column | Type | Description |
|--------|------|-------------|
| id | uuid | Primary key |
| action_type | enum | proposal_created | vote_certified | system_update | rule_change | admin_action | identity_event |
| actor_ref | text | Role or system actor (not the secret voter) |
| payload_hash | text | Hash of the action payload |
| prev_hash | text | Hash of the previous entry (hash chain) |
| signature | text | Cryptographic signature |
| created_at | timestamptz | Append time |

## Notes

Implements FR-060, FR-065. Append-only, hash-chained, publicly verifiable (NFR-015). Voting privacy preserved: ballot content never logged here.
