## Columns

| Column | Type | Description |
|--------|------|-------------|
| id | uuid | Primary key (internal civic identity) |
| public_handle | text | Non-identifying public participation handle |
| citizenship_status | enum | citizen | revoked | suspended |
| legal_identity_hash | text | Salted hash binding to a verified legal identity; raw identifiers never stored here |
| status | enum | active | inactive | revoked |
| created_at | timestamptz | Identity creation time (also emitted to audit_log) |

## Notes

Core civic identity. Implements FR-001, FR-002. Government identifiers, address, and voting history are NEVER stored or exposed here (NFR-006).
