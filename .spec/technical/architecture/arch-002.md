## Overview

Every public action follows a single end-to-end lifecycle from problem identification through outcome evaluation. This pipeline gives each proposal a consistent, traceable path and defines where eligibility, expertise, budget, and audit checks apply.

## Details

Stages: Problem Identification -> Proposal Creation -> Public Review & Expert Analysis -> Budget & Impact Assessment -> Voting -> Implementation -> Public Monitoring -> Outcome Evaluation (archive). Gates: impact scope assigned before voting (FR-008); support threshold before development (FR-017); funding information before voting (FR-037); eligibility from sphere of impact at voting (FR-041); tracking and outcome evaluation after approval (FR-046, FR-048). Deadlock at any stage is handled by the staged resolution framework (FR-034). Governance effort scales with decision impact (decision escalation).
