## Description

As a constitutional review body, I can block proposals that violate protected rights so that fundamental rights are protected from majorities.

## Acceptance Criteria

- [ ] A defined set of protected rights cannot be altered by ordinary votes
- [ ] Proposals violating protected rights are flagged in review
- [ ] Constitutional review can block rights-violating proposals

## Notes

- Implements requirements: FR-058
- Technical aspects: ADR-010
- Protects against tyranny of the majority.
