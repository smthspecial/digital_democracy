## Description

As a system, I can expire and re-review competency periodically so that no permanent expert class emerges.

## Acceptance Criteria

- [ ] Competency carries an expiry and review schedule (e.g. 2-5 years)
- [ ] Inactive contributors lose active status
- [ ] Re-validation is required to retain status

## Notes

- Implements requirements: FR-026
- Technical aspects: ADR-009
- Competency is not permanent.
