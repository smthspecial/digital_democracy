## Description

As a identity registrar, I can revoke identities under qualifying conditions so that the register stays accurate while abuse is prevented.

## Acceptance Criteria

- [ ] Revocation workflows exist for death, loss of citizenship, and proven fraud
- [ ] Each revocation requires documented justification and is logged
- [ ] A revoked identity can no longer participate

## Notes

- Implements requirements: FR-006
- Technical aspects: ADR-005
- Revocation is logged and justified.
