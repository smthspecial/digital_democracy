## Description

As a citizen, I can inspect the immutable public action log so that corruption can be exposed but not erased.

## Acceptance Criteria

- [ ] All governance actions are written to an append-only log
- [ ] Records are cryptographically secured and publicly verifiable
- [ ] Tampering or deletion is detectable; voting privacy is preserved

## Notes

- Implements requirements: FR-060
- Technical aspects: ADR-005, NFR-015
- Immutable audit trail.
