## Description

As a oversight board member, I can require multiple approvals before any participation is disabled so that no single administrator can silence a citizen.

## Acceptance Criteria

- [ ] Disabling an identity requires multiple independent approvals
- [ ] No single role can complete the action alone
- [ ] All approvers are recorded in the action log

## Notes

- Implements requirements: FR-007
- Technical aspects: ADR-011, NFR-008
- Anti-capture safeguard on the identity layer.
