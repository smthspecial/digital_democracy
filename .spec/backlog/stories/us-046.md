## Description

As a system, I can require elevated thresholds for constitutional change so that boundary changes are deliberate and broadly supported.

## Acceptance Criteria

- [ ] Constitutional changes require a supermajority
- [ ] Elevated participation thresholds apply
- [ ] Extended deliberation and expert assessment are mandatory

## Notes

- Implements requirements: FR-059
- Technical aspects: ADR-010, ADR-011
- Constitutional change is hard by design.
