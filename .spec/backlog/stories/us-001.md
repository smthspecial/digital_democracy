## Description

As a citizen, I can register and obtain a single verified civic identity so that I can participate in governance as a unique, eligible person.

## Acceptance Criteria

- [ ] I can complete identity verification through an accepted method (national ID, passport, or government credential)
- [ ] The system rejects any attempt to create a second active identity for me
- [ ] My government identifiers are never publicly exposed

## Notes

- Implements requirements: FR-001, FR-002
- Technical aspects: ADR-003, NFR-002, NFR-006
- Foundation for one-person-one-vote.
