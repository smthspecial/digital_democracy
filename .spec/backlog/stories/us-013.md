## Description

As a proposer, I can submit competing proposals and develop alternatives so that citizens choose among real options, not a single proposal.

## Acceptance Criteria

- [ ] Multiple proposals can be linked to one problem
- [ ] A proposal that reaches threshold enters a development phase
- [ ] Participants can submit alternatives, modifications, and cost estimates

## Notes

- Implements requirements: FR-018, FR-019
- Technical aspects: ADR-006
- The proposal marketplace.
