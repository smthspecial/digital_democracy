## Description

As a citizen, I can challenge a scope assignment I believe is wrong so that incorrect eligibility decisions can be corrected.

## Acceptance Criteria

- [ ] I can file a scope challenge on a proposal
- [ ] An independent review body resolves the dispute publicly
- [ ] A pending challenge can pause the affected voting phase

## Notes

- Implements requirements: FR-010, FR-011
- Technical aspects: ADR-004
- Due process for eligibility.
