## Description

As a citizen, I can apply for domain competency with evidence so that my demonstrated expertise becomes visible per domain.

## Acceptance Criteria

- [ ] I can apply for competency in a specific domain with evidence
- [ ] Automated verification of credentials runs where possible
- [ ] My application is publicly reviewable before approval

## Notes

- Implements requirements: FR-021, FR-022
- Technical aspects: NFR-004
- Competency is domain-specific and auditable.
