## Description

As a citizen, I can see reputation signals separate from competency so that I can judge the track record of contributors.

## Acceptance Criteria

- [ ] Reputation is tracked independently of competency level
- [ ] Positive and negative factors are transparent
- [ ] Reputation never alters voting weight
