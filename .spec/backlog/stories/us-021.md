## Description

As a participant, I can define the constraints a solution must satisfy before debating solutions so that we find common ground before arguing methods.

## Acceptance Criteria

- [ ] A proposal records agreed constraints before solution design
- [ ] Solutions are evaluated against the agreed constraints
- [ ] Constraint sets are public

## Notes

- Implements requirements: FR-029
- Technical aspects: ADR-012
- Constraint-based governance.
