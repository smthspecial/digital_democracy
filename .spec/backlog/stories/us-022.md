## Description

As a citizen, I can declare my desired outcomes separately from policy arguments so that my priorities feed proposal development.

## Acceptance Criteria

- [ ] I can record preferences as desired outcomes
- [ ] Preferences are captured before solution proposals
- [ ] Declared preferences inform proposal generation

## Notes

- Implements requirements: FR-030
- Technical aspects: ADR-012
- Preference declaration.
