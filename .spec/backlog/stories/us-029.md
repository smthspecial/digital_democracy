## Description

As a citizen, I can allocate participatory budget within categories so that spending reflects collective priorities.

## Acceptance Criteria

- [ ] I can allocate preferences across partitioned budget categories
- [ ] Allocation happens within categories, not one undifferentiated pool
- [ ] Aggregated allocations are public and inform the budget

- Implements requirements: FR-038, FR-039
- Technical aspects: NFR-005
