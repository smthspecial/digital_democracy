## Description

As a independent auditor, I can review identity-creation events so that I can detect fraud or manipulation without accessing private data.

## Acceptance Criteria

- [ ] Every identity-creation event appears in an append-only log
- [ ] I can review events through public interfaces without operator cooperation
- [ ] Anomalous creation patterns are surfaced

## Notes

- Implements requirements: FR-005
- Technical aspects: ADR-005, NFR-004
- Independent oversight of the identity layer.
