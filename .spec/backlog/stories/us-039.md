## Description

As a citizen, I can receive a small number of randomized civic assignments so that governance work is fairly distributed.

## Acceptance Criteria

- [ ] I receive a limited number of review/audit assignments
- [ ] Assignments are weighted by impact, competency, workload, and randomness
- [ ] Assignment frequency adjusts automatically

## Notes

- Implements requirements: FR-050
- Technical aspects: ADR-009, NFR-014
- Randomized civic assignment.
