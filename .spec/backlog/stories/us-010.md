## Description

As a citizen, I can submit a problem with description, area, and evidence so that issues I experience enter the governance pipeline.

## Acceptance Criteria

- [ ] I can create a problem entry with description, affected area, and evidence
- [ ] The issue is assigned a candidate impact scope
- [ ] The issue becomes publicly visible

## Notes

- Implements requirements: FR-015
- Technical aspects: NFR-010
- Entry point of the lifecycle.
