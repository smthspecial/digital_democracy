## Description

As a proposer, I can advance a proposal once it reaches its support threshold so that only proposals with real backing reach a vote.

## Acceptance Criteria

- [ ] A proposal advances only after meeting its support threshold
- [ ] Thresholds scale with the population of the impact scope
- [ ] Threshold values are public and configurable

## Notes

- Implements requirements: FR-017
- Technical aspects: ADR-009
- Mitigates interest-group capture.
