## Description

As a citizen, I can support, comment on, and add evidence to issues so that important problems gain visible momentum.

## Acceptance Criteria

- [ ] I can support an issue and my support is counted auditably
- [ ] I can comment and attach evidence
- [ ] Support totals are public

## Notes

- Implements requirements: FR-016
- Technical aspects: NFR-005
- Public endorsement of issues.
