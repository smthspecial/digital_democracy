## Description

As a citizen, I can revoke my delegation at any time and have it auto-expire so that no permanent delegate accumulates power.

## Acceptance Criteria

- [ ] I can revoke a delegation at any time
- [ ] Delegations expire automatically after a defined period
- [ ] No delegate holds delegated authority permanently

## Notes

- Implements requirements: FR-057
- Technical aspects: ADR-008, NFR-008
- No permanent delegates.
