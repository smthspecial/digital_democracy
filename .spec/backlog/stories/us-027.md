## Description

As a citizen, I can explore the real-time public budget ledger so that I can see where public money comes from and goes.

## Acceptance Criteria

- [ ] Revenue sources and allocations are visible in real time
- [ ] Approved expenditures and project costs are traceable
- [ ] The ledger is publicly accessible and understandable

## Notes

- Implements requirements: FR-036
- Technical aspects: ADR-005, NFR-005
- Budget transparency.
