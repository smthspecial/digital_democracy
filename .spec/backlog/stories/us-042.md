## Description

As a citizen, I can serve in randomly selected audit pools so that continuous, distributed oversight is possible.

## Acceptance Criteria

- [ ] I can be randomly selected for budget review, project verification, or outcome evaluation
- [ ] Domain experts support audits without controlling them
- [ ] Audit findings are published

## Notes

- Implements requirements: FR-055
- Technical aspects: ADR-009, NFR-004
- Random audit pools.
